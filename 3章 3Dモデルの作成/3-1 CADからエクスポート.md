# 3-1でやること  
CADソフトでモデリングしたものを、3DCG系ソフトで扱えるようにするために、3Dモデルの一般的な形式.stlに変換してエクスポートします。

## SOLIDWORKSのエクスポート設定
`ファイル>指定保存` からファイルの種類`.stl`を選択。  
下の`オプション`を開き、単位を`m`に設定する。
