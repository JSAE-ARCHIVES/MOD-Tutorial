# 3Dモデルの作成
3章 3Dモデルの作成はbasicパートとadvancedパートに分かれています。  
- basicパート：最低限のクオリティでCADからAssettoCorsaにコンバートすることを目標にしています。  
- advancedパート：見た目のクオリティを上げるためのテクニックを紹介しています。
